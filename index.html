Alguns conhecimentos que tenho

HTML

É uma linguagem baseada em marcação, onde marcamos os elementos para definir quais informações a página vai exibir.

CSS

É uma linguagem de folha de estilo composta por "camadas", criado com o propósito de estilizar as páginas HTML.

JavaScript

É uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma.

MySQL

o MySQL é um poderoso sistema de gerenciamento de banco de dados relacional de código aberto, amplamente utilizado em aplicativos da web para armazenar e recuperar dados.

Figma

é uma plataforma de design de interface do usuário baseada em nuvem que permite a colaboração em tempo real entre equipes de design. Ele oferece uma ampla gama de recursos de design, incluindo ferramentas de desenho vetorial, prototipagem e animação, e é conhecido por sua capacidade de colaboração e compartilhamento de projetos em tempo real.

Github

é uma plataforma de hospedagem de código-fonte baseada na web que permite que desenvolvedores e equipes de desenvolvimento colaborem em projetos de software. Ele fornece ferramentas para gerenciamento de repositórios de código-fonte, rastreamento de problemas, gerenciamento de projetos e integração com outras ferramentas de desenvolvimento.
